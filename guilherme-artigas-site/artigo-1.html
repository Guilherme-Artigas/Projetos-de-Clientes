O que são Tags em HTML 5?

O que é HTML?

Bom pessoal dando uma rápida introdução ao assunto TAG em HTML é trivial entender e saber o que é HTML 5 e qual a sua função nesse vasto mundo de linguagens e tecnologias.


HTML 5 é uma linguagem de marcação padrão para se criar páginas ou um conjunto de páginas formando web sites. HTML significa HyperText Markup Language (traduzindo para o nosso querido português brasileiro: Linguagem de Marcação de HiperTexto), um documento HTML é formado por diversas TAGS ou como acabamos encontrando em vários materiais de apoio, elementos. TAG e elementos são a mesma coisa dentro da HTML.


Uma TAG/ELEMENTO em HTML se inicia com aberturas e fechamento em sua grande maioria, sempre iniciando-a com o sinal matemático de menor (<)TAG(>) envolvendo-a com o sinal matemático maior, exemplo: <p>. Acabamos de abri uma TAG, porém, como já mencionado aqui nesse texto a grande maioria das TAGS são constituídas por abertura (<p>) e fechamento (</p>), é isso o fechamento das TAGS são como a abertura, porém com uma barra antes do nome da TAG/ELEMENTO, o conteudo entre elas sera exibido pelos navegadores cumprindo assim a sua função, vou deixar abaixo alguns exemplos de TAGS.


<article></article>

<aside></aside>

<audio></audio>

<body></body>

<form></form>
